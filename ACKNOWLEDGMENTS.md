# Acknowledgments

PhysicsCourt builds on several public research models and libraries:

- V-JEPA 2, DINOv2, SAM 2, and CoTracker from Meta / FAIR.
- Depth Anything V2 from the Depth Anything authors.
- PyTorch, Transformers, Hugging Face Hub, OpenCV, NumPy, SciPy, scikit-learn, and related open-source tools.

This repository does not redistribute pretrained model weights. The scripts
download checkpoints from Hugging Face or the relevant upstream source into the
user's local cache. Third-party checkpoints, packages, and datasets remain
under their own licenses and terms. Please check those upstream licenses before
redistributing weights or using the models in another setting.

The PhysicsCourt source code, project-written documentation, and benchmark
orchestration code are released under the MIT License unless a file says
otherwise.
